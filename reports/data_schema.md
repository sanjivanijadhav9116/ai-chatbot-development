# Data Schema

Dataset Name: Chatbot Dataset

Format: CSV

Columns:

1. Question (Text)
   - User input or query from user

2. Answer (Text)
   - Response generated by chatbot

3. Category (Text)
   - Type of query like Greeting, FAQ, General, etc.

Number of Records:
11

Storage Method:
Local Storage

Stored File:
data/chatbot_dataset.csv

Purpose:
To store cleaned chatbot training data for model development and testing.
