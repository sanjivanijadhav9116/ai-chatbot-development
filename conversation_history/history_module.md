# Conversation History Module

## Objective

The objective of this module is to store and retrieve previous interactions between the user and the AI chatbot.

## Description

The Conversation History Module stores user questions and AI responses. It allows the chatbot to save, load, and display previous conversations.

## Activities

1. Store user prompts.
2. Store AI responses.
3. Save conversation data in a JSON file.
4. Retrieve previous conversation history.
5. Display previous questions and answers.
6. Add new conversations to the history.

## Data Storage

Conversation data is stored in the `conversation_history.json` file.

Each conversation record contains:

- Question: The prompt entered by the user.
- Answer: The response generated by the AI chatbot.

## Python Module

The `history.py` file provides the following functions:

### 1. Load Conversation History

The `load_conversation_history()` function reads previously stored conversations from the JSON file.

### 2. Save Conversation

The `save_conversation()` function saves a new question and answer to the conversation history.

### 3. Display Conversation History

The `display_conversation_history()` function displays all stored questions and answers.

## Example

Question: How are you?

Answer: I'm doing great! How can I help you today?

## Expected Functionality

The module should be able to:

- Store new questions and answers.
- Retrieve previously stored conversations.
- Display conversation history.
- Maintain multiple conversation records.

## Conclusion

The Conversation History Module helps the AI chatbot maintain a record of previous interactions. This feature allows conversations to be stored and retrieved, improving the user experience.
